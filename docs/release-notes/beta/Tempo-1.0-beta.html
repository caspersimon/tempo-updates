Tempo Beta 1.0

Build 1

This is the first public beta build of Tempo, the on-device macOS batch transcription app for video creators and podcasters.

What to expect

  • Signed with Developer ID and notarized by Apple.
  • Delivered on the Sparkle beta update channel for future in-app updates.
  • Designed for Apple Silicon Macs running macOS 14 or later.
  • Still in beta, so features and polish may change between releases.

If you run into rough edges, crashes, or workflow confusion, please send that feedback along before the stable release.
